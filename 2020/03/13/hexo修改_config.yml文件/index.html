    修改代码我建议使用微软的 VS code

    以下的步骤也是在VS code中进行的

    一. 打开blog文件

    首先我们得先安装VS code这个软件，百度vs code就能找到官网

    或者复制这个链接 https://code.visualstudio.com/

    点击下载链接即可下载，可能会有点慢，请耐心等待。

    安装的时候，第二步的时候请把是否创建桌面快捷方式打上勾勾

    vs code 默认可能会是英文的，可以自行去百度搜搜教程，博主已经忘了，记得安装个中文包重启就好了。

    话不多说，我们去下载代码的位置将代码在VS code中打开

    博主的上个教程中将代码存储的位置是D盘中的hexo文件夹下

    如果您没有设置下载路径的话，可以去c:/用户/你的电脑名(笔记本的话会是品牌名，比如HP)/

    然后在里面找一个叫做blog的文件夹。

    如果设置了路径，跟着上一个教程照着做的话，

    那么打开d:/hexo下会有一个文件夹叫做blog，

    我们将blog文件夹按住拖动到VS code中去

    打开后会有一堆文件

    我们能操作的不多，所以大部分我们都不需要去动它，让它紧紧呆着就好。

    我们首先要了解的，也是最重要的文件 _config.yml

    我们单击打开，又是一堆看不懂的英文，不要着急（着急也没有用）

    让博主来一一为你翻译并告诉你它们各自的用法：

    YML是什么

    YAML (YAML Aint Markup Language)是一种标记语言，通常以.yml为后缀的文件，是一种直观的能够被电脑识别的数据序列化格式，并且容易被人类阅读，容易和脚本语言交互的，可以被支持YAML库的不同的编程语言程序导入，一种专门用来写配置文件的语言。可用于如： Java，C/C++, Ruby, Python, Perl, C#, PHP等。

    总结：yml是一种标记语言，一种专门用来写配置文件的语言。

    其他的我们就不关心了，我们要知道的，重点

    yml 使用空格的缩进表示层级关系，空格数目不重要，只要是左对齐的一列数据，都是同一个层级的

    缩进时不允许使用Tab键，只允许使用空格。

    # 这是一条注释(中间需要打一个空格！)

    yml文件不允许使用tab键，层级关系使用空格！！！

    到这里，我们就记住这一点就好了，好了，开始翻译。
