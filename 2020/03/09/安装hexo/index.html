注意，用hexo搭建的博客是静态的网站。

最开始，需要先安装两个依赖

node.js 和 git

node.js网址： https://nodejs.org/zh-cn/

git网址：https://git-scm.com/

选择好跟你电脑一样的平台点击下载并安装。
